# Travel OS 1.0 RC4

## Added

- Smart Mode that adapts to planning, flight, transit, morning, afternoon, and evening contexts
- Trip progress / Day X of 14
- Travel Timeline on Home and Today
- First-run traveler onboarding
- Larger-text preference
- Apple Wallet Suica preference reminder
- Photo Quest
- Seasonal planning page
- Traveler-specific packing and favorites
- Smart reminder cards
- Loading skeletons
- Button/gesture motion polish
- Haptic attempts where the browser supports vibration
- Improved offline service-worker strategy
- Settings page
- Updated 14-day route beginning October 27, 2026

## Intentionally not included

- Native app rebuild
- Live location sharing
- Built-in AI chat
- Unverified future event dates

Those remain postponed to keep the PWA stable and predictable.
