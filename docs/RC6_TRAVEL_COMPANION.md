# Travel OS 1.0 RC6 — Travel Companion

## New modules
- Travel Companion command center
- Professional time-based Daily Timeline
- Reservation Assistant with local status tracking
- Weather-aware itinerary and Rain Mode decisions
- Live-transit launchpad with mapping and official rail-status links
- Context-aware Packing Intelligence
- Planned-vs-actual Budget Dashboard
- Searchable Map Explorer
- Smart Daily Brief
- Local Memory Book with print / Save as PDF support

## Data and dates
- Current trip dates: October 26–November 11, 2026
- Route: Tokyo → Hakone → Kyoto → Osaka → Tokyo
- 128 Food Explorer restaurant candidates remain integrated

## Important limitations
- Platform, delay, exit, and train composition information comes from external live services and requires connectivity.
- The Smart Daily Brief is generated deterministically from your trip data; it does not send private data to an external AI service.
- Restaurant awards, operating hours, prices, menus, reservation rules, and dietary handling require re-verification before booking.
