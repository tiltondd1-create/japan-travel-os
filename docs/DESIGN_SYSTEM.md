# Design System

## Direction

Calm, readable, trustworthy.

Reference mood:
Apple Maps × Notion × Japanese minimalism.

## Color Tokens

| Token | Hex | Use |
|---|---:|---|
| Background | `#08100B` | Main app background |
| Card | `#FFF8EB` | Primary cards |
| Primary | `#6F1D1B` | Brand / important buttons |
| Accent Red | `#A63A3A` | Highlights |
| Sage | `#DCE8D4` | Comfort / Hilda / confidence |
| Blue | `#DCEAF7` | Weather / transit |
| Lavender | `#E8E2F3` | Calm / Hilda |
| Gold | `#E8C878` | Reminders / highlights |

## UI Rules

- Use large touch targets.
- Keep emergency screens high contrast.
- Prefer fewer options on Hilda pages.
- Use the same wording for repeated actions.
- Keep Money, Suica, SOS, and Lost Mode one-tap accessible.
- Avoid adding major features until content/testing is complete.

## Component Rules

- Hero card: top-level screen identity.
- Action tile: navigates to another screen.
- Info card: readable content.
- Alert card: important reminder.
- Chip: status/category.
- Big button: one main action.
