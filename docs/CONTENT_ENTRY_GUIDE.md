# Content Entry Guide

Use `workbook/Travel_OS_Master.xlsx` as the active Google Sheets source.

## Fill these first

### 1. Hotels — Critical
Fill:
- Hotel name
- Address
- Japanese address if available
- Phone number
- Nearest station
- Google Maps link
- Check-in / check-out
- Notes for Hilda / taxi card

### 2. Maps — Critical
Add links for:
- Hotels
- Major stations
- Restaurants
- Attractions
- Bathrooms / department stores
- Meetup points
- Airports

### 3. Apple Wallet Suica — Critical
Confirm:
- Each traveler has Apple Wallet Suica plan
- Express Transit note
- Backup physical card note
- Battery bank reminder
- Top-up/reload guidance

### 4. Emergency / Lost Mode — Critical
Fill:
- Hotel address card
- Police / ambulance info
- Embassy / clinic info if desired
- Help phrases
- Meetup plan if separated

### 5. Restaurants — High
For each place:
- Name
- City
- Area / nearest station
- Cuisine
- Must order
- Price range
- Reservation needed?
- Hours if known
- Google Maps link
- Notes

### 6. Phrases — High
Prioritize:
- Lost
- Taxi / hotel
- Restaurant ordering
- Konbini
- Transit
- Medical / emergency
- Polite basics

### 7. Packing — High
Include:
- Battery bank
- Medication
- Passport
- Chargers / adapters
- Umbrella / jacket
- Daily carry
- Apple Wallet Suica setup

## Good enough rule

Do not try to make everything perfect first. Start by filling what would be painful to not have offline:
hotel, maps, emergency, transit, Suica, and phrases.
